INTRODUCCION DE GIT Git es una herramienta de código abierto que la inicio Linux, El creador. Una ventaja de Git es que con un solo programa puede ser trabajado simultáneamente por varios desarrolladores Git y Github se basan en el concepto de repositorio cada uno de los desarrolladores tendrán su propio repositorio de forma local, pero tendremos en control cada vez que cambiemos en el código Podremos acceder a versiones anteriores del mismo y podemos revertir cambios Su importancia es que nos permite tener control de todos los cambios de nuestro código, El control de versiones es una de las tareas más importantes para desarrollar un proyecto y mientras que GITHUB es algo de desarrolladores repositorios de software

Historia de GIT y GITHUB Git es un sistema de control de versiones que fue creada por Linux pensando en la eficiencia Y la confiabilidad del mantenimiento de versiones de aplicaciones cuando estas tienen un gran número de archivos de código de fuente, Desde su nacimiento en 2005 Git ha evolucionado y madurado para ser fácil usar y aun conservar estas cualidades iniciales. GITHUB el 24 de febrero de 2009 el equipo de github anuncio en una charla en las oficinas centrales de yahoo! que es su primer año habían logrado acumular repositorios El 9 de julio github anuncio que los beneficios de github habían crecido un 300% anualmente desde 2008 Siendo provechoso durante casi todo el año.

COMO INSTALAR GIT 1 Ve a https://git-scm.com y descarga el paquete de instalación 2 Ahora debes elegir la opción según la arquitectura de tu computadora a 32bits o 62bits 3 Ahora debes ejecutar el archivo descargado y veras una pantalla 4 ahora elige la carpeta donde ubicaras los archivos de git 6 asegúrate de tener seleccionado git bash que es la herramienta trabajaremos y con esto se terminara la instalación 7 ahora tendrás disponible git bash desde tu lista de programas.
